# 大学物理讲义网页设计规格书

**日期**: 2026-04-10
**项目**: 大学物理讲义在线网站
**目标**: 将 LaTeX 讲义转换为可浏览的静态网页

---

## 1. 项目概述

将 LaTeX 编写的大学物理讲义（book1.tex）转换为静态网页，包含公式高亮、答案折叠/展开、目录导航功能。本地生成静态文件，通过 git 推送到服务器部署。

## 2. 技术方案

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| 转换工具 | Python + pandoc | 读取 LaTeX 文件，转换为 HTML |
| 样式 | 纯 CSS | 公式色块、折叠展开 |
| 交互 | 原生 JavaScript | 目录导航、答案交互 |
| 部署 | 静态文件 + git | 本地构建，git push 到服务器 |

## 3. 页面结构

```
/
├── index.html          # 封面页
├── ch1.html           # 第1章 质点运动学
├── ch2.html           # 第2章 ...
├── ch3.html
├── ch4.html
├── ch5.html
├── ch6.html
├── css/
│   └── style.css      # 统一样式
├── js/
│   └── main.js        # 交互逻辑
└── asset/              # 图片资源
```

## 4. 功能规格

### 4.1 公式高亮
- 公式使用带背景色的圆角方块显示
- 背景色: 浅绿色 (#e8f5e9)
- 左边框: 深绿色 (#2e7d32)
- 圆角: 6px
- 内边距: 12px 16px

### 4.2 答案折叠/展开
- **例题答案**: 默认展开，带"收起"按钮
- **补充练习答案**: 默认折叠，带"展开"按钮
- 点击按钮切换状态，过渡动画 0.3s

### 4.3 目录导航
- 左侧固定目录栏
- 点击章节标题跳转到对应页面
- 页面内显示当前章节的小节目录
- 目录高亮当前阅读位置

### 4.4 一键更新脚本
- 文件: `build.py`
- 运行命令: `python build.py`
- 功能:
  1. 读取 latex/ 目录下的所有章节文件
  2. 使用 pandoc 转换为 HTML
  3. 注入 CSS 样式和 JavaScript
  4. 输出到根目录

## 5. 样式规格

### 5.1 配色方案（与 LaTeX 讲义保持一致）

| 模块 | 章节 | 主色调 | 色值 |
|------|------|--------|------|
| 力学 | ch1-ch4 | MainGreen | #16a085 |
| 振动和波 | ch5-ch6 | MainPurple | #8e44ad |

**说明**: 答案（补充练习答案）直接放在对应章节的补充习题下方，无需单独配色

公式高亮使用各章节对应的主色调作为左边框色。

### 5.2 字体
- 中文: "Noto Serif SC", "Source Han Serif SC", serif
- 英文/公式: "KaTeX_Math", "Times New Roman", serif

### 5.3 布局
- 左侧目录栏: 260px 宽度
- 内容区: 自适应，最大 800px
- 响应式: 移动端隐藏目录，改为汉堡菜单

## 6. 验收标准

1. 运行 `python build.py` 成功生成所有 HTML 文件
2. 公式有明显的高亮效果
3. 例题答案默认展开，可收起
4. 补充练习答案默认折叠，可展开
5. 目录点击可跳转到对应章节
6. 页面在浏览器中正常显示，无报错
